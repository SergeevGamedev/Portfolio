This is a demo project showcasing the usage of DOTween, Odin Inspector, UniRx, UniTask, UI Toolkit and adherence to SOLID principles

### Quick Start

1. Install plugins:
- [DOTween](https://assetstore.unity.com/packages/tools/animation/dotween-hotween-v2-27676?srsltid=AfmBOooE_IJT0iCWH1FOYpgKDyRlnK3JQeKyUZ6THpj5JbOtPeLD56cX)
- [Odin Inspector](https://assetstore.unity.com/packages/tools/utilities/odin-inspector-and-serializer-89041?srsltid=AfmBOooF5zoy2Yfs1_lf1buBsHnyPqYi-xw4AUasfTcvCsKL6zpfYhhL)
- [UniRx](https://assetstore.unity.com/packages/tools/integration/unirx-reactive-extensions-for-unity-17276?srsltid=AfmBOor4-HA-26ySWj2-tZihuWzgzSFcrCDKagtSK2V56Blh8y5SFHpa)
- [UniTask](https://github.com/Cysharp/UniTask)

2. Download or Clone the Repository
3. Open the Scenes/Demo.unity
4. Select the AchievementManager prefab in the scene hierarchy
5. Use the buttons in the AchievementManager inspector:
- Kill Enemy
- Collect Coin
- Complete Level
